# Agent Call Examples

A gallery of `agent[T](prompt)` calls in the Scala REPL.

## Simple value lookup

```scala
scala> val year: Int = agent[Int]("the year Alan Turing was born")
val year: Int = 1912
```

## Filtering a list using context

```scala
scala> val xs = List(0, 1, 2, 4, 7, 9, 10)
val xs: List[Int] = List(0, 1, 2, 4, 7, 9, 10)

scala> agent[List[Int]]("filter the prime number from xs")
[agent] filtering primes from xs
val res1: List[Int] = List(2, 7)
```

## Structured data into a case class

```scala
scala> case class Person(name: String, born: Int, field: String)
// defined case class Person

scala> val turing: Person = agent[Person]("basic info about Alan Turing")
val turing: Person = Person(
  name = "Alan Turing",
  born = 1912,
  field = "Computer science"
)
```

## Sorting with semantic knowledge

The agent supplies facts the program does not know.

```scala
scala> val films = List("Inception", "Citizen Kane", "Parasite", "Metropolis")
val films: List[String] = List(
  "Inception",
  "Citizen Kane",
  "Parasite",
  "Metropolis"
)

scala> agent[List[String]]("films sorted by release year, oldest first")
val res1: List[String] = List(
  "Metropolis",
  "Citizen Kane",
  "Inception",
  "Parasite"
)
```

## ADT / recursive structure

The agent picks the right constructors of a sum type.

```scala
scala> enum Json:
         case Obj(fields: Map[String, Json])
         case Arr(items: List[Json])
         case Str(s: String)
         case Num(n: Double)
         case Bool(b: Boolean)
         case Nul
// defined class Json

scala> val sample: Json = agent("a small JSON tree describing a user with name, age, and two hobbies")
val sample: Json = Obj(
  Map(
    "name" -> Str("Alice"),
    "age" -> Num(30.0),
    "hobbies" -> Arr(List(Str("reading"), Str("hiking")))
  )
)
```

## Serializing a user defined ADT

The agent knows the shape of `sample` from its type and reuses it.

```scala
scala> val sampleStr: String = agent("convert variable `sample` into json string")
[agent] serializing sample to JSON string
val sampleStr: String = "{\"name\":\"Alice\",\"age\":30.0,\"hobbies\":[\"reading\",\"hiking\"]}"
```
